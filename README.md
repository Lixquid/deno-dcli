<h1><a href="https://dcli.lix.cx">Deno CLI Tools</a></h1>

Quick scripts, run with Deno, to be used on the command line.

Usage: `deno run https://dcli.lix.cx/<tool>`

A list of tools is available at https://dcli.lix.cx. The list of scripts is
also embedded as a comment in the source code at the bottom of the page, and
can be seen with `curl https://dcli.lix.cx`.
